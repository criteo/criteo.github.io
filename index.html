Open Source @ Criteo

Criteo is committed to contributing to the open source community.

We use ample open-source software internally (Hadoop, RabbitMQ, Graphite, Couchbase, Cassandra, git, gitlab, GNU/Linux, HAProxy, just to name a few). We also contribute to open software we use with bug fixes and improvements. Additionally, we publish tools of our own.

Here are our Github repositories: One for all Criteo maintained open source projects, and another for all Criteo maintained Chef cookbooks
